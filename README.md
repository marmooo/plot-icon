# Plot Icon

[An icon plotting game to connect dots](https://marmooo.github.io/plot-icon/).

## Installation

1. `gitn clone vendor/ repos.lst` (see [gitn](https://github.com/marmooo/gitn))
2. `bash build-svg.sh`
3. `deno run --allow-read --allow-write listup.js`
